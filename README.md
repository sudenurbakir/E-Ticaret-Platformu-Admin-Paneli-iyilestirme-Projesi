# E-Ticaret-Platformu-Admin-Paneli-iyilestirme-Projesi

PROJE ADI: E-Ticaret Platformu Admin Paneli İyileştirme Projesi

Hazırlayan: Sudenur Bakır

Tarih: Temmuz 2026

 1. Giriş / Proje Özeti

Bu proje, bir e-ticaret platformunun admin panelinin iş analizi ve iyileştirme çalışmalarını kapsamaktadır. Mevcut sistemde yaşanan yavaşlık, manuel işlemler ve yetersiz raporlama sorunları hedeflenmiştir.

İş Problemleri
- Sipariş onay süreci ortalama 8 dakika sürüyor.
- Stok güncellemeleri manuel yapılıyor ve hatalar oluşuyor.
- Yöneticiler farklı raporları farklı sistemlerden alıyor.
- Admin arayüzü karmaşık ve yeni çalışanlar için zor.

Önerilen İyileştirmeler
- Süreç otomasyonu
- Tek dashboard üzerinden tüm raporlara erişim
- Rol bazlı yetki sistemi
- Gerçek zamanlı stok takibi
- İyileştirilmiş arama ve filtreleme

Kullanılan Yöntemler
- Gereksinim Toplama (Stakeholder Interview)
- As-Is / To-Be Süreç Modelleme (BPMN)
- User Stories
- Wireframing
